---
title: "Install Popovers via Swift Package Manager or CocoaPods"
description: "Add Popovers to your Xcode project using Swift Package Manager or CocoaPods. Requires iOS 14+ and Swift 5.5+. Zero external dependencies."
---

Popovers is available through Swift Package Manager (recommended) and CocoaPods. Both methods take only a few minutes. Once you have the library added, a single `import Popovers` statement gives you access to the full API.

## Prerequisites

- Xcode 13 or later
- iOS 14.0+ deployment target
- Swift 5.5+

## Add the package

<Steps>
  <Step title="Choose your package manager">
    Select Swift Package Manager if you are starting fresh or prefer Xcode's built-in tooling. Use CocoaPods if your project already has a Podfile.

    <CodeGroup>

    ```text Swift Package Manager
    https://github.com/aheze/Popovers
    ```

    ```ruby CocoaPods (Podfile)
    pod 'Popovers'
    ```

    </CodeGroup>
  </Step>

  <Step title="Add the dependency">
    <Tabs>
      <Tab title="Swift Package Manager">
        1. Open your project in Xcode.
        2. Go to **File → Add Package Dependencies**.
        3. Paste the package URL into the search field:

        ```text
        https://github.com/aheze/Popovers
        ```

        4. Select a version rule (e.g. **Up to Next Major Version**).
        5. Click **Add Package** and confirm the target you want to link against.
      </Tab>
      <Tab title="CocoaPods">
        1. If you don't have a Podfile yet, run `pod init` in your project directory.
        2. Open your Podfile and add the dependency:

        ```ruby
        pod 'Popovers'
        ```

        3. Save the file, then run:

        ```bash
        pod install
        ```

        4. Open the generated `.xcworkspace` file instead of `.xcodeproj` from now on.
      </Tab>
    </Tabs>
  </Step>

  <Step title="Import the module">
    At the top of any Swift file where you want to use Popovers, add the import:

    ```swift
    import Popovers
    ```

    In SwiftUI files you will typically import both frameworks:

    ```swift
    import SwiftUI
    import Popovers
    ```
  </Step>
</Steps>

## Verify the installation

Build your project (`Cmd+B`). If there are no errors, Popovers is ready to use. You can confirm the module is available by typing `Popover` in any Swift file — Xcode's autocomplete will show the `Popover` struct and the `.popover(present:)` modifier.

<Tip>
  Popovers has zero external dependencies, so adding it will not pull in any additional packages.
</Tip>

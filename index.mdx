Popovers is a Swift library that makes it effortless to present custom popovers, menus, alerts, and overlays in iOS apps. With a single `.popover` modifier in SwiftUI or a simple `present()` call in UIKit, you can display any view above your app's content — fully customizable in position, animation, and dismissal behavior.

## Why Popovers?

Popovers takes a modern approach to overlays in iOS — built on a SwiftUI core with full UIKit support, zero dependencies, and an expressive API that handles everything from simple tooltips to complex multi-popover flows.

## Get started

<Steps>
  <Step title="Install the library">
    Add Popovers to your Xcode project using Swift Package Manager with the URL `https://github.com/aheze/Popovers`, or add `pod 'Popovers'` to your Podfile.
  </Step>
  <Step title="Import and present">
    Import `Popovers` and attach the `.popover(present:)` modifier to any SwiftUI view — or create a `Popover` instance and call `present()` in UIKit.
  </Step>
  <Step title="Customize">
    Use the `attributes` closure to configure position, animation, dismissal behavior, accessibility, and more to match your app's design.
  </Step>
</Steps>

<Note>
  Popovers requires iOS 14+ and has zero external dependencies. It's built on a SwiftUI core for a lightweight footprint.
</Note>

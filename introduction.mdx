Popovers is a Swift library that lets you present any view above your app's main content on iOS. Whether you need a tooltip anchored to a button, a picture-in-picture style floating panel, a custom menu, or a full-screen overlay, Popovers gives you a single consistent API that works in both SwiftUI and UIKit — with no external dependencies.

## What you can build

- Custom alerts, confirmation dialogs, and action sheets
- Tooltips and onboarding coach marks anchored to UI elements
- Picture-in-picture style floating panels the user can drag between positions
- Context menus that match (or exceed) the look of the system `Menu`
- In-app notifications and banners
- Multi-step tutorial overlays

## Capabilities at a glance

- **Present any view** — pass any SwiftUI `View` as the popover's content or background
- **Absolute and relative positioning** — attach to a source view or float freely like a PIP window
- **Multiple simultaneous popovers** — display several at once with smooth animated transitions between them
- **Fully customizable attributes** — control position, presentation animation, dismissal mode, rubber banding, background blocking, and accessibility behavior through the `Popover.Attributes` struct
- **Interactive drag behavior** — popovers can be dragged off screen or between anchor positions
- **VoiceOver support** — built-in accessibility with configurable focus shifting and dismiss button
- **Ready-made templates** — ship faster with `Templates.Menu`, `BackgroundWithArrow`, `Container`, `CurveConnector`, and more
